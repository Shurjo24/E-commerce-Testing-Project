# E-Commerce Website Testing Project

## Project Overview
This project demonstrates practical manual Software Quality Assurance (SQA) testing experience using an e-commerce demo website.

The objective of this project was to practice:
- Manual Testing
- Test Case Writing
- Bug Reporting
- QA Documentation
- Functional Testing
- Usability Testing
